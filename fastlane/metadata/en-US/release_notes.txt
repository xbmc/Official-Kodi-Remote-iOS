Features:
- Added to main menu: Files, Add-ons and Settings
- Support CEC volume control (see app settings)
- Start app via URL scheme (e.g. "kodiremote://192.168.1.2")
- Support app shortcuts (via long pressing app icon)
- List and load user profiles

Bugfixes:
- Fix "Error 153" for embedded youtube videos
- Fix custom buttons for several settings types
- Do not truncate dates in library view

Improvements:
- Higher resolution for all icons and images
- App settings reachable via shortcut from app
- "Start in view" option independent of translations
- Re-connect on restart/wakeup after quit/shutdown/hibernate/suspend
- Settings show labels (e.g. "Movie plot") instead of numbers
- Setting sliders show units (e.g. "ms")
- iPad custom button view as popover
- Movable servers in list
- Support Romanian
- Support Bosnian
- Translation updates for several languages
- Many more small improvements...

Remark: "Start in view" needs to be set again after updating to this version.
